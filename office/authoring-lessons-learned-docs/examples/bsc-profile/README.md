# Example: BSC Onboarded Profile for Lessons Learned

This folder contains the tokenized base template built by onboarding Business Science Corporation (BSC) as an organization.

## Contents
- `assets/lessons-learned-base-template.docx` — BSC's own real Lessons Learned document, copied and tokenized: cover title/date converted to `{{PROJECT_CODE}}` / `{{COVER_DATE}}`, identity table tokenized, and "Negatives/Challenges" standardized to "Challenges".
