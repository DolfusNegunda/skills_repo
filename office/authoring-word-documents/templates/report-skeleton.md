# Report Skeleton (heading map)

Copy this heading tree into a new document, apply the mapped styles, then fill.

```
[Title]                          → Title style
[Subtitle / date / author]       → Subtitle style

Contents                         → (auto TOC field)

1. Executive Summary             → Heading 1
2. Background                     → Heading 1
   2.1 Context                    → Heading 2
   2.2 Scope                      → Heading 2
3. Findings                       → Heading 1
   3.1 [Theme]                    → Heading 2
       Figure 1. [caption]        → Caption
4. Analysis                       → Heading 1
5. Recommendations                → Heading 1
6. Next Steps / Action Plan       → Heading 1
Appendix A. [data / methodology]  → Heading 1
Glossary                          → Heading 1
References                        → Heading 1
```

Front matter: Title, Subtitle, TOC.
Body: sections 1–6.
Back matter: appendices, glossary, references.
Footer: page number (field). Header: document title + confidentiality mark.
