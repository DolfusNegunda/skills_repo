# Productivity Skills

Individual working competencies: planning, task and time management, personal
knowledge, note-taking, decision journaling, goals, and learning. Every skill here
follows the house style in [../skill-builder/SKILL.md](../skill-builder/SKILL.md).

## Design philosophy

Pure **procedure and judgment skills** — no scripts. Unlike the other categories
(which produce work *products*), these help a person (or an agent acting for one)
*work well*: capture and prioritize, protect focus, retain knowledge, and learn. Each
adds the method and the specific traps (overpacked days with no buffer, competing task
lists, vague non-actionable tasks, knowledge hoarding with no retrieval, verbatim
notes never revisited, journaling the choice but not the rationale, passive rereading
instead of recall). They lean on the [../reasoning](../reasoning/README.md) methods for
prioritizing and deciding.

## The skills

**Planning & doing**
| Skill | Focus |
|---|---|
| [planning-your-day](planning-your-day/SKILL.md) | Most-important tasks, time-blocking, protected focus, buffer. |
| [managing-tasks](managing-tasks/SKILL.md) | One trusted GTD-style system: capture → clarify → organize → review. |
| [managing-time](managing-time/SKILL.md) | Deep work, batching, single-tasking, timeboxing, saying no. |
| [breaking-down-tasks](breaking-down-tasks/SKILL.md) | Big/vague task → small concrete steps with an obvious first action. |

**Knowledge & notes**
| Skill | Focus |
|---|---|
| [managing-personal-knowledge](managing-personal-knowledge/SKILL.md) | Capture, organize (PARA), link, retrieve — a findable second brain. |
| [taking-notes](taking-notes/SKILL.md) | Cornell/outline/atomic methods; meaning over transcript; process into the system. |

**Judgment & growth**
| Skill | Focus |
|---|---|
| [keeping-decision-journals](keeping-decision-journals/SKILL.md) | Log context/rationale/expected outcome; review to calibrate. |
| [setting-and-tracking-goals](setting-and-tracking-goals/SKILL.md) | Outcome vs process goals, a leading measure, tracking, review cadence. |
| [planning-learning](planning-learning/SKILL.md) | Target proficiency, prerequisites, resources, milestones, practice + feedback. |
| [studying-effectively](studying-effectively/SKILL.md) | Active recall, spaced repetition, Feynman, interleaving over rereading. |

## How they compose

- **Run your week:** [planning-your-day](planning-your-day/SKILL.md) draws from
  [managing-tasks](managing-tasks/SKILL.md); [breaking-down-tasks](breaking-down-tasks/SKILL.md)
  turns each big item actionable; [managing-time](managing-time/SKILL.md) protects the execution.
- **Build lasting knowledge:** [taking-notes](taking-notes/SKILL.md) →
  [managing-personal-knowledge](managing-personal-knowledge/SKILL.md).
- **Grow deliberately:** [setting-and-tracking-goals](setting-and-tracking-goals/SKILL.md) →
  [planning-learning](planning-learning/SKILL.md) → [studying-effectively](studying-effectively/SKILL.md);
  [keeping-decision-journals](keeping-decision-journals/SKILL.md) calibrates judgment over time.
