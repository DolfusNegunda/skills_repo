# SWOT + TOWS Template

## SWOT (evidence each factor)

```
              HELPFUL                        HARMFUL
        +-----------------------------+-----------------------------+
INTERNAL| STRENGTHS                   | WEAKNESSES                  |
        | - [factor + evidence]       | - [factor + evidence]       |
        +-----------------------------+-----------------------------+
EXTERNAL| OPPORTUNITIES               | THREATS                     |
        | - [factor + evidence]       | - [factor + evidence]       |
        +-----------------------------+-----------------------------+
```

## TOWS (turn factors into strategy)

| | **Opportunities (O)** | **Threats (T)** |
|---|---|---|
| **Strengths (S)** | **SO** — use strengths to seize opportunities | **ST** — use strengths to counter threats |
| **Weaknesses (W)** | **WO** — fix weaknesses to pursue opportunities | **WT** — minimize weaknesses, avoid threats |

## Selected strategies
1. [SO/WO/ST/WT] — [concrete action, owner]
2. ...

Rules: S/W internal, O/T external; evidence every factor; the TOWS strategies are
the deliverable, not the four lists.
